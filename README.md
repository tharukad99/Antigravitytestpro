# CakeyNuts Website

Welcome to the CakeyNuts website project! This is a modern, responsive, and premium website designed for a high-end cake business.

## Features
- **Modern Design**: Built with Tailwind CSS for a sleek, responsive look.
- **Multi-page Layout**: Includes Home, Menu, About, and Contact pages.
- **Interactive Elements**: Mobile-friendly navigation, hover effects, and smooth scrolling.
- **Premium Aesthetics**: Gold, brown, and cream color palette suitable for a luxury bakery.

## How to Run
Since this project uses the Tailwind CSS CDN, there is no need to install Node.js or run any build commands.

1. Simply open `index.html` in any modern web browser (Chrome, Edge, Firefox).
2. Navigate through the links to view different pages.

## Customization
- **Images**: Currently using high-quality Unsplash placeholders. distinct images can be added to an `assets/images` folder.
- **Colors**: The color palette is defined in the `<script>` tag in the `<head>` of each HTML file under `tailwind.config`.

Enjoy your new website!
